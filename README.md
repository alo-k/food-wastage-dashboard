# Food Wastage Management Dashboard

## 📌 Overview

This project is a Streamlit-based interactive dashboard designed to analyze and visualize food wastage data. It helps connect food providers with receivers by offering insights into food availability, claims, and distribution patterns.

## 🚀 Features

* Interactive filters (City, Food Type, Meal Type)
* Key performance indicators (KPIs)
* Data visualizations using Plotly
* Analysis of food claims and provider contributions
* Contact information for food providers

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Plotly

## 📊 Data Insights

The dashboard provides insights such as:

* Top contributing food providers
* Most common food types and meal categories
* Claim status distribution (Completed, Pending, Cancelled)
* Food availability across locations

## ▶️ How to Run

```bash
streamlit run app.py
```
